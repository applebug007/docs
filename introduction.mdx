## Authentication

To generate an x-api-key:

1. Go to the [Developer Page](https://www.scam.ai/home/developer).

<img height="200" src="/images/developer-settings.png" />

2. Click on **Create API Key** in the page.

<img height="200" src="/images/create-api-key.png" />

3. Copy the API key and use it for calling the API.
