## Understanding the Threats

### Deepfakes Explained
Deepfakes are AI-generated images where a person's face is replaced with someone else's face or digitally manipulated. They're created using deep learning algorithms that can:

- **Face Swapping**: Replace one person's face with another's in images
- **Expression Transfer**: Change facial expressions while keeping the identity
- **Age/Gender Modification**: Alter apparent age or gender characteristics
- **Synthetic Face Generation**: Create completely artificial faces that look real
- **Feature Manipulation**: Alter specific facial features or characteristics

**Common Uses in Fraud**:
- Identity theft and impersonation
- Fake profile pictures on social media
- Fraudulent ID documents
- Creating false evidence
- Social engineering attacks
- Catfishing and romance scams

### Detection Challenges
Modern deepfake technology is increasingly sophisticated, making detection challenging:

- **High-Quality Models**: Advanced AI generates highly realistic faces
- **Minimal Artifacts**: Fewer visible signs of manipulation
- **Real-time Generation**: Fast processing enables immediate use
- **Accessibility**: User-friendly tools make creation easier
- **Constant Evolution**: New techniques bypass existing detection methods

### Blended Fakeness Score
Our model outputs a **blended fakeness score** ranging from 0.0 to 1.0:

- **Score calculation**: Uses ensemble methods combining multiple detection techniques
- **Higher scores**: Indicate stronger evidence of deepfake manipulation
- **Lower scores**: Suggest the image is likely authentic
- **Confidence scoring**: Each detection includes a confidence measure for reliability

### Deepfake Detection

Analyze images for face manipulation and deepfake content:

```bash
POST http://3.80.33.105:5000/predict
```

**Headers:**
- `x-api-key`: Your API key (required for authentication)

**Request:**
- **Content-Type**: `multipart/form-data`
- **Body**: Form data with `file` field containing the image to analyze

**Supported Image Formats:**
- JPEG (.jpg, .jpeg)
- PNG (.png) 
- Other common image formats

### Authentication

All prediction requests require an API key in the header:

```bash
x-api-key: your_api_key_here
```

### Error Responses

The API returns specific error codes for different scenarios:

| Status Code | Error | Description |
|-------------|-------|-------------|
| 401 | Unauthorized | Missing or invalid API key |
| 400 | Bad Request | No file provided or invalid file type |
| 422 | Unprocessable Entity | No face detected in the image |
| 200 | Success | Analysis completed successfully |

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `faces` | array | Array of detected faces in the image |
| `faces[].blended_fakeness_score` | number | Deepfake probability score (0.0 - 1.0) |
| `faces[].face_region` | object | Bounding box coordinates of detected face |
| `faces[].confidence` | number | Face detection confidence score |
| `processing_time_ms` | number | Time taken to process the request |
| `image_metadata` | object | Original image information |

### Understanding Fakeness Scores

| Score Range | Interpretation | Recommendation |
|-------------|---------------|----------------|
| 0.0 - 0.2 | Likely authentic | Low risk - appears to be real |
| 0.2 - 0.5 | Uncertain | Medium risk - manual review recommended |
| 0.5 - 0.8 | Likely manipulated | High risk - likely deepfake content |
| 0.8 - 1.0 | Highly likely deepfake | Very high risk - strong deepfake indicators |

```python Python
import requests

# API Configuration
BASE_URL = "http://3.80.33.105:5000"
API_KEY = "your_api_key_here"

def health_check():
    """Check if the API is healthy"""
    response = requests.get(f"{BASE_URL}/health")
    return response.json()

def detect_deepfake(image_path):
    """Detect deepfake in image file"""
    headers = {
        "x-api-key": API_KEY
    }
    
    with open(image_path, 'rb') as image_file:
        files = {
            'file': image_file
        }
        
        response = requests.post(
            f"{BASE_URL}/predict",
            headers=headers,
            files=files
        )
    
    return response.json()

# Example usage
if __name__ == "__main__":
    # Check health
    health = health_check()
    print(f"API Health: {health}")
    
    # Detect deepfake
    try:
        result = detect_deepfake("suspicious_image.jpg")
        
        if "faces" in result:
            for i, face in enumerate(result["faces"]):
                score = face["blended_fakeness_score"]
                print(f"Face {i+1}: Fakeness Score = {score:.3f}")
                
                if score > 0.8:
                    print("  ⚠️  HIGH RISK: Likely deepfake")
                elif score > 0.5:
                    print("  ⚠️  MEDIUM RISK: Possibly manipulated")
                elif score > 0.2:
                    print("  ℹ️  LOW RISK: Uncertain, review recommended")
                else:
                    print("  ✅ AUTHENTIC: Likely real")
        else:
            print(f"Error: {result}")
            
    except Exception as e:
        print(f"Request failed: {e}")
```

```javascript JavaScript
const BASE_URL = 'http://3.80.33.105:5000';
const API_KEY = 'your_api_key_here';

// Health Check
async function healthCheck() {
  try {
    const response = await fetch(`${BASE_URL}/health`);
    return await response.json();
  } catch (error) {
    console.error('Health check failed:', error);
    throw error;
  }
}

// Deepfake Detection
async function detectDeepfake(imageFile) {
  const formData = new FormData();
  formData.append('file', imageFile);

  try {
    const response = await fetch(`${BASE_URL}/predict`, {
      method: 'POST',
      headers: {
        'x-api-key': API_KEY
      },
      body: formData
    });
    
    if (!response.ok) {
      const error = await response.json();
      throw new Error(`API Error: ${error.detail}`);
    }
    
    return await response.json();
  } catch (error) {
    console.error('Detection failed:', error);
    throw error;
  }
}

// Example usage with file input
document.getElementById('imageInput').addEventListener('change', async (event) => {
  const file = event.target.files[0];
  if (!file) return;

  try {
    // Check API health first
    const health = await healthCheck();
    console.log('API Status:', health);

    // Analyze image
    const result = await detectDeepfake(file);
    
    if (result.faces && result.faces.length > 0) {
      result.faces.forEach((face, index) => {
        const score = face.blended_fakeness_score;
        console.log(`Face ${index + 1}: Fakeness Score = ${score.toFixed(3)}`);
        
        let riskLevel = 'AUTHENTIC';
        if (score > 0.8) riskLevel = 'HIGH RISK - Likely deepfake';
        else if (score > 0.5) riskLevel = 'MEDIUM RISK - Possibly manipulated';
        else if (score > 0.2) riskLevel = 'LOW RISK - Uncertain';
        
        console.log(`  Risk Level: ${riskLevel}`);
      });
    } else {
      console.log('No faces detected in image');
    }
    
  } catch (error) {
    console.error('Analysis failed:', error.message);
  }
});
```
